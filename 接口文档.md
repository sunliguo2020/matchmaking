# 接口返回的基本格式

```json

{
    "code":200,
    "data":[]
    "msg":""
}
```



##  幸福案例 

### 请求接口

http://127.0.0.1:8000/api/anli/

### 请求方式

GET

### 请求参数

- page 页数
- size 每页数量

### 响应示例

http://127.0.0.1:8000/api/anli/?page=3&size=2
